# Task 16 - Time Series Forecasting

## Dataset Used
Walmart Store Sales Dataset

## Objective
Perform time series analysis and forecast future weekly sales using Python.

## Steps Performed
- Converted Date column to datetime
- Aggregated total weekly sales
- Visualized sales trend
- Split data into train and test (time-based split)
- Applied Exponential Smoothing model
- Generated 12-week forecast
- Evaluated model using MAE and MAPE
- Exported forecast results to CSV

## Deliverables
- task16_forecasting.ipynb
- forecast_output.csv
- forecast_report.txt
